Etiquetas Semanticas

En HTML5 actualmente se han implementaados una variedad de etiquetas semanticas las cuales son de utilidad para el desarrollo web ya que indican que contenido tiene esta misma; Tomemos como ejemplo la etiqueta Div o B, el div o division es para dividir un contenido de otro y la etiqueta B es mas precisa que la etiqueta STRONG (es la que un titulo o texto se resalte mas que los otros), la etiqueta B significa que el elemento que estara adentro sea negrita.
Las etiquetas semanticas son las que nos indican que contenido conforma nuestra pagina web una vez se halla enderizado por el navegador.
